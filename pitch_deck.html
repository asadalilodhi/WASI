The Problem with Food Delivery

  • Traditional Chatbots Fail: Rigid decision trees cannot handle how humans actually speak.
  • App Fatigue: Customers don't want to download a new app for every local restaurant.
  • Language Barriers: Existing bots fail completely at understanding mixed languages like Roman-Urdu.

"Samajh nahi aya, kya aap menu se kuch order karna chahte hain?"

- Every bad chatbot ever

WASI Slide 2

The Solution: WASI

A native WhatsApp experience powered by a Mesh of Specialized AI Agents.


🗣️ Natural Language

Understands messy Roman-Urdu, slang, and implied context.

🔄 Dynamic State

AI maps conversation directly to strict JSON database schemas.

⚡ Multi-Agent

Menu Agent, Delivery Agent, and Returning User Agent working together.

👀 Human-in-the-loop

A live React dashboard for receptionists to step in when needed.

WASI Slide 3

Architecture & Tech Stack

  • Band AI Cloud: Central telemetry, observability, and webhook orchestration.
  • LangGraph (Python): State machine defining agent workflows and memory.
  • Groq / AIML API: Lightning-fast LLM inference locking AI into strict JSON Mode.
  • React & Vite: Live dashboard for restaurant receptionists.
